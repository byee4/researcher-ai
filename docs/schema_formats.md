# Schema Formats

This document catalogs schema-like types used by `researcher-ai`.
It is generated by `scripts/generate_schema_catalog.py`.

## Format Conventions

- Primary schema format: Pydantic `BaseModel` objects serialized to JSON.
- Internal orchestration state format: `TypedDict` structures (Python dict shape contracts).
- Parser-side extraction schemas are prefixed with `_` and are LLM structured-output contracts.

- Total schemas cataloged: `80`
